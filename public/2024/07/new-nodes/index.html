Rescue IPMI

The IPMI in the nodes were set to ipv6 only and I didnâ€™t know the password. To set them to DHCP was easy in the bios, however you canâ€™t set the password from the bios. To reset the passwordâ€™s I resorted to using the IPMItool. Due to some security changes in ESXi8 it was necessary to install ESXi7 to reset the password.

To do this I did a vanilla install of ESXi then uploaded the ipmitool to the datastore before running the below command

/opt/ipmitool/ipmitool raw 0x30 0x40

This set the IPMI to factory defaults and took about 30 seconds to complete. From there I could configure as required.
